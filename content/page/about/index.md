## Who am I?
仮想の世界と現実の世界の両方で、写真撮影やものづくりをしています。  
コンピュータと向き合い、実装で殴られ、仕様で転びながら、動くところまで持っていくのが好きです。  
時には、VR空間や現実の風景を切り取りっています。
これまでに取り組んだ制作物や技術的な試みの一部は、ブログ記事としてまとめています。  
VRChatではアバターを用いて活動しており、SNSでは主にVRCアバターの写真を投稿しています。  
また、フィルムカメラを用いた写真撮影も行っています。  
その他のSNSや連絡先については、[リンクページ](/content/page/links/index.md)をご参照ください。


## 技術スタック

### 低レイヤ / 基盤

#### Assembly language
CASLⅡおよび、自作32bit CPU向けに設計した独自アセンブリ言語を扱っています。  
命令セット設計、レジスタ操作、メモリ管理を通じて、CPUの基本的な動作原理を理解しています。

#### C
初めて学んだプログラミング言語です。 
構造体、列挙型、ポインタを用いたメモリ操作を理解し、用途に応じた実装が可能です。

#### 電子工作
Raspberry Piと各種センサを用いた簡易的な回路構成・制御の経験があります。  
はんだ付けや実機デバッグを含む、プロトタイピングレベルの作業が可能です。

### 高級言語 / アプリケーション

#### Python
IoTサーバやCLIツールの開発・運用で使用しています。  
Linux / Raspberry Pi環境における常駐プログラムやスクリプトの作成が可能です。

#### C#（Unity）
Unityを用いた2D・3Dゲーム開発の経験があります。  
コンポーネント設計やシーン管理を理解した上で実装しています。

#### Java
授業およびチーム開発で使用しています。  
クラス設計、継承、ポリモーフィズムなど、オブジェクト指向を用いた実装が可能です。

### OS / 開発環境

#### Debian系 Linux
Raspberry Pi OS / Raspbianを中心に使用しています。  
SSH経由での操作や、Bashを用いた運用・管理しています。

#### Ubuntu
主にWSL環境で使用しています。  
Bashを用いた開発・検証環境として利用しています。

#### Windows
デスクトップ環境として日常的に使用しています。  
PowerShellやコマンドプロンプトによる基本的な操作が可能です。
